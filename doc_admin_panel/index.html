About the Files

GoTaxi App is a online taxi system which run under Android platform that used for your own online taxi application. This application created by Android for client side and then codeIgniter for Admin side. Run under Android platform which is the most popular operating system in the world. Using this application you can save your money and time in creating application for your own online taxi application.

Getting Started

Before doing installation, you need to do the following things :
 
Have a web hosting to store admin panel files and Mysql database, to run this admin panel, web hosting that you use must have the following requirements :
  • PHP Version 5.6 or 7.0
  • Support Mysqli
  • Hosting Server use hostinger.com (we highly recommend using hostinger.com)
  • The gotaxi script does not work on the Panel PLESK and VPS. Support CPANEL HOSTINGER.COM!!
 
Please login on your cpanel account

  • enter your username and your password

 

Database Creation

1. Login to your cpanel hosting and open MySQL Databases
2. Create database and give database name like you want, as example : 'Gotaxi_app_db’
3. Create user for your database
4. Add user to your database and check ALL PRIVILEGES

Import Database .sql file

1. Open phpMyAdmin and select your database
2. Import file database gotaxi_app_db.sql  from download package inside 'admin_panel' folder
 
 
 

Upload PHP Scripts and Database Configuration

1. open File Manager via cpanel or ftp server, upload gotaxi_app.zip inside 'admin_panel' folder to your folder destination on your hosting, then extract it

2. to change configuration please folder see :

A) admin : public_html/admin/application/config/database.php

B) api : public_html/api/app/config/database.php

C) application : public_html/application/config/database.php

D) frontend : public_html/frontend/application/config/database.php

E) partner : public_html/partner/application/config/database.php

3. you must edit this data with your own data.

 
 

Setting Base URL CodeIgniter

to change configuration please base URL is set in the config.php file like :

    2) Now open Driver.php in folder ( ../api/app/controllers/Driver.php ) and change the following:
    Replace the text on line 339
